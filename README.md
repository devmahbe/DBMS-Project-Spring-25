# 🚨 Crime & Complaint Reporting System

An intelligent **Crime & Complaint Reporting System** developed as part of our **Database Management Systems (DBMS)** course project for Spring 2025. 
This system allows users to report crimes, track complaint statuses, and assists authorities in maintaining structured and accessible crime records.

---

## 📁 Project Structure
## 🧠 Features

✅ Register & Login (Users / Admins)  
✅ File complaints with category & location  
✅ View and update complaint status  
✅ Admin dashboard for monitoring reports  
✅ MySQL database with raw SQL queries  
✅ Built using Express.js, TailwindCSS, EJS

---

## 🛠️ Technologies Used

- **Backend**: Node.js, Express.js
- **Frontend**: HTML, Tailwind CSS, EJS
- **Database**: MySQL
- **Tools**: WebStorm, Git, GitHub
